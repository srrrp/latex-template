## LaTeX template for note taking

This template can be used for taking notes. It uses latex for compiling tex files which is rendered as pdf file.

### Setup

- Install latex (for macos using homebrew):  `brew install --cask mactex-no-gui`
- Install [VimTex](https://github.com/lervag/vimtex) to edit latex files using vim
- Additionally install [zathura](https://github.com/zegervdv/homebrew-zathura) (command line based pdf viewer)
